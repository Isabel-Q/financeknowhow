# FinanceKnowHow

FinanceKnowHow 是一个桌面端财务知识产品，面向没有财务背景的 CEO 和经营负责人。

当前 MVP 聚焦中国大陆与新加坡，核心目标是把高频财务议题翻译成经营语言，并用一个可检索、可筛选、可回链官方来源的桌面知识操作台承载。

## 当前能力

- CEO 视角总览：先讲现金、报表、预算、经营节奏
- 中国 / 新加坡知识库：税务、会计、治理、薪酬等核心主题
- 经营日历：按月、按季、按年与事件触发节点
- 财务术语表：把财务黑话翻译成老板能直接理解的语言
- 官方来源入口：从条目详情直接打开监管或税务机构页面

## 技术栈

- `Tauri 2`
- `React 19`
- `TypeScript`
- `Vite`

## 本地运行

```bash
npm install
npm run tauri dev
```

仅构建前端：

```bash
npm run build
```

## 内容说明

- 内容是面向经营者的理解框架，不构成正式税务、审计或法律意见。
- 中国与新加坡条目已内置一批高频主题与官方来源链接。
- 校验日期写在产品顶部，当前版本记录为 `2026-05-13`。
